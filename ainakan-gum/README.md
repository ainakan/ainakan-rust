# ainakan-gum [![docs.rs](https://docs.rs/ainakan-gum/badge.svg)](https://docs.rs/ainakan-gum)

Rust bindings for [Ainakan Gum](https://github.com/ainakan/ainakan-gum).

## Before Installing

- Build Ainakan or download the [latest supported package](https://github.com/ainakan/ainakan/releases/)
- Move `ainakan-gum.h` and `libainakan-gum.a` into `/usr/local/include` and `/usr/local/lib` (or a more appropriate folder that Rust can detect)

Or: use the `auto-download` feature to install Ainakan.

See the documentation for usage instructions.
