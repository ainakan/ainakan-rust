# ainakan [![docs.rs](https://docs.rs/ainakan/badge.svg)](https://docs.rs/ainakan)

Rust bindings for [Ainakan](https://ainakan.re).

## Before Installing

- Build Ainakan or download the [latest supported package](https://github.com/ainakan/ainakan/releases/)
- Move `ainakan-core.h` and `libainakan-core.a` into `/usr/local/include` and `/usr/local/lib` (or a more appropriate folder that Rust can detect)

Or: use the `auto-download` feature to install Ainakan.

See the documentation for usage instructions.
